# Types of Aid
  * [FIU's Golden Promise](https://onestop.fiu.edu/finances/types-of-aid/fius-golden-promise/index.html)
This program guarantees 100% tuition and fees for Florida residents with a Student Aid Index (SAI) of 0 or less.
  * [Scholarships](https://onestop.fiu.edu/finances/types-of-aid/scholarships/index.html)
Awards based on merit and/or need which do not have to be repaid.
  * Awards based on financial need which do not have to be repaid.
  * Funds which are borrowed and must be repaid with interest.
  * [Work-Study Jobs](https://onestop.fiu.edu/finances/types-of-aid/work-study-jobs/index.html)
Part-time jobs for students with financial need allowing them to help pay for educational expenses.
